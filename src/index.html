Introduccion a los Vectores

Los vectores son una herramienta utilizada para representar fenomenos fisicos en un sistema de referencia, por ejemplo un carro moviendose a cierta velocidad que se mueve a la derecha, puede ser estudiado con un vector velocidad `vecV` que se mueve en direccion `hati`. Existen muchos tipos de vectores, entre ellos la mencionada velocidad, aceleracion, posicion, fuerza, campos, etc. En esta seccion veremos una introduccion al tema e iremos profundizando a medida que avanzemos.

Figura 1: Caracteristicas de un vector.

Para representar un vector, lo hacemos mediante con una flecha arriba del nombre (`vecA`), el cual tiene una magnitud (`abs(vecA)`) y componentes (`Axhati` y `Ayhatj`), en la siguiente figura se muestra una representacion grafica de lo que se acaba de hablar.

Figura 2: Componentes de un vector

En este caso el vector `vecV` puede ser representado de la siguiente manera:
`vecV = Axhati + Ayhatj`
Donde las letras "`hati`" y "`hatj`" representan los vectores unitarios, es decir, los vectores por lo general se situan en el plano de coordenadas X-Y, donde la cada eje tiene asociado un vector unitario que solo indica la direccion en la que apunta el vector, los cuales tiene un tamaño de uno. Su utilidad recae en que podemos hacer uso del teorema de pitagoras para obtener la magnitud y direccion del vector conociendo las componentes.

Suma de Vectores

Los vectores pueden sumarse de manera grafica y analitica.

Suma Grafica de vectores

Metodo punta y cola

Si se desea sumar dos vectores `vecA` y `vecB`, tomamos el final de `vecA` y lo unimos con el principio de `vecB`, finalmente unimos el principio de `vecA` con el final de `vecB`. Figura 7: Metodo punta y cola para suma grafica de vectores.

Metodo del paralelogramo

Si se desea sumar dos vectores `vecA` y `vecB`, unimos el origen de ambos vectores y luego trazamos lineas paralelas, en el caso del `vecA`, nos ubicamos en su final y trazamos una linea paralela al vector `vecB`, hacemos lo mismo con el otro, nos ubicamos en su final y trazamos una linea paralela, el punto de encuentro se une con el origen de ambos vectores para obtener el resultante. Figura 8: Metodo del paralelogramo para suma grafica de vectores.

Suma Analitica de vectores

En general la suma de vectores es la suma de sus componentes `hati`, `hatj` y `hatk`, por ejemplo:
Si tenemos `vecA = 2hati + 3hatj` y `vecB = hati + 4hatj`, la suma de ambos sera un vector que llamaremos `vecC`
`vecC = (2+1)hati + (3+4)hatj`
`vecC = 3hati + 7hatj`

Funciones Trigonometricas

Son funciones que relacionan los lados de un triangulo recto entre si, es muy utilizado en el analisis vectorial. Figura 9: Resumen de las funciones trigonometricas. Consideremos el siguiente vector. Figura 9: Vector en el plano 2D.
Ese vector `vecR = Rxhati + Ryhatj`, puede ser expresado mediante las funciones trigonometricas como:
`Rx = abs(R)*cos(theta)`
`Ry = abs(R)*sen(theta)`
Donde la magnitud se puede obtener mediante el teorema de pitagoras. Figura 10: Teorema de pitagoras.
`abs(vecR) = sqrt(Rx^2 + Ry^2)`
Para obtener su direccion `theta` utilizamos la funcion tangente. tg`theta = frac{Ry}{Rx}`

Producto Escalar

Dados dos vectores `vecA` y `vecB`, un producto escalar `vecA * vecB` se define como el producto de sus módulos por el coseno del ángulo `theta` que forman, cuyo resultado es un numero escalar. Esta operacion representa la proyeccion de un vector sobre otro.

Figura 11: Producto escalar.

Siendo `vecA = axhati + ayhatj + azhatk` y `vecB = bxhati + byhatj + bzhatk`, dos vectores, m y n dos escalares, podemos resumir las propiedades del producto escalar de la siguiente forma.

  • `vecA * vecB = vecB * vecA`
  • `vecA * (vecB + vecC) = vecA * vecB + vecA * vecC`
  • `m (vecA * vecB) = (mvecA)*vecB = (vecA * vecB) m`
  • `hati*hati = hatj*hatj = hatk*hatk = 1`
  • `hati*hatj = hatj*hatk = hatk*hati = 0`
  • `vecA*vecB = (axhati)(bxhati) + (ayhatj)(byhatj) + (akhatk)(bzhatk)`

Producto Vectorial

Dados dos vectores `vecA` y `vecB`, un producto vectorial `vecA xx vecB` se define como el producto de sus módulos por el seno del ángulo `theta` que forman, cuyo resultado es un vector. Esta operacion representa un vector perpendicular a ambos.

Figura 12: Producto vectorial.

Siendo `vecA = axhati + ayhatj + azhatk` y `vecB = bxhati + byhatj + bzhatk`, dos vectores, m y n dos escalares, podemos resumir las propiedades del producto vectorial de la siguiente forma.

  • `vecA xx vecB = -(vecB xx vecA)`
  • `vecA xx (vecB + vecC) = vecA xx vecB + vecA xx vecC`
  • `m (vecA xx vecB) = (mvecA)xxvecB = (vecA xx vecB) m`
  • `hatixxhati = hatjxxhatj = hatkxxhatk = 0`
  • `hatixxhatj = hatjxxhatk = hatkxxhati = 1`
  • `vecAxxvecB = abs([i,j,k],[ax,ay,az],[bx,by,bz]) = abs([j,k],[ay,az],[by,bz])*hati + abs([i,k],[ax,az],[bx,bz])*hatj +abs([i,j],[ax,ay],[bx,by])*hatk = (ay*bz - az*by)hati + (ax*bz - az*bx)hatj + (ax*by - ay*bx)hatk`
